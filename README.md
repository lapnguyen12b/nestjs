# nestjs

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start
```
# watch mode
$ npm run start:dev
```

```bash
# unit tests
$ npm run test
```

## Linting

```bash
$ npm run lint
```

## Formatting

```bash
$ npm run format
```

```bash
npm run build
npm run typeorm migration:run
```
